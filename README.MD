# Blackjack

Juego de Blackjack desarrollado con HTML, CSS y JavaScript.  

## Características

- Juego completo con reglas tradicionales de Blackjack.
- Interfaz limpia y responsiva.
- Contador de puntos actualizado en tiempo real.
- Lógica completa: reparto de cartas, turno del jugador, turno del crupier, detección de blackjack, empates, victorias y derrotas.
- Sin dependencias externas.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
